  • RED|BLACK

    This was a project I did in my CS316 class. It utilizes HTML, CSS and JavaScript along with the API given by the teacher to create a game.
    This game fetches a deck of cards and randomly chooses five cards from the deck. The cards are initially flipped and are turned when user selects one of the button.
    A player wins if the color selected is the majority. This also has option to allow user to reset the stats, show cheats- the number of colors remaining in the deck and a progress bar showing number of remaining cards in deck.
