Below are some projects I have been working on:

    RED|BLACK : A web-based game created using HTML, CSS and JavaScript that incorportes external APIs(provided by CS316 teacher). The program fetches a deck of cards from API and then randomly chooses five cards from the deck. The player is then allowed to choose one of two options- red or black. A player wins if the color selected is the majority (3 or more). This game integrates Semantic UI framework to create a responsive and visually appealing user interface with clear segmentation for game, statistics, history, and cheats.

    SIMPLE CHAT PROGRAM : Multithreaded chat application that enables simultaneous message exchange between two users using Python's socket and threading libraries.     PERSONAL WEBSITE : Personal website built using HTML, CSS and JavaScript featuring dedicated sections for “Home”, “Contact”, and “About Me” to effectively showcase my skills, background, and personal projects. Integrates CSS and JavaScript to implement interactive elements, such as hover effects, zoom on click, and transition on click.

    SAT Solver : SAT solver that uses DPLL algorithm. The program utilizes unit propagation, which simplifies the CNF (Conjunctive Normal Form) formula by deducing assignments for variables that appear in unit clauses. The algorithm then searches for unassigned variables within the formula and then recursively assigns truth values (either true or false) to these variables. If an assignment leads to a conflict where no clauses can be satisfied, the algorithm backtracks by unassigning the last variable and exploring other potential assignments. This recursive process continues until a satisfying assignment is found or all possibilities have been exhausted.
