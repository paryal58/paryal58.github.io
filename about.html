    SIMPLE CHAT PROGRAM : Multithreaded chat application that enables simultaneous message exchange between two program windows using Python's socket and threading libraries.     SIMPLE CALCULATOR : A simple calculator app created using React. It utilizes javascript's math.js to perform neccessary calculations.

    SAT SOLVER : SAT solver that uses DPLL algorithm. The program utilizes unit propagation, which simplifies the CNF (Conjunctive Normal Form) formula by deducing assignments for variables that appear in unit clauses. The algorithm then searches for unassigned variables within the formula and then recursively assigns truth values (either true or false) to these variables. If an assignment leads to a conflict where no clauses can be satisfied, the algorithm backtracks by unassigning the last variable and exploring other potential assignments. This recursive process continues until a satisfying assignment is found or all possibilities have been exhausted.
